## Overview
Dropdowns present a list of options from which a user can select one option, or several.

<UsageGuidelines guidelines={[

        description:"If you don’t want to keep the user’s choice visible, use the Menu component instead."
      }
    },

### Dropdown inside a form
A classic dropdown presents options a user needs to choose from.
<Canvas>
  <Story name="Dropdown inside a form">
    {() => {
      const oprions = [
        { value: "Sometimes", label: "Sometimes" },
        { value: "No", label: "No" },
        { value: "Yes", label: "Yes" },
      ];
      return (
        <div className="">
          <h6>Are you usually a Dark mode person?</h6>
          <Dropdown
            defaultValue={[oprions[0]]}
            placeholder="Placeholder text here" 
            options={oprions} 
            className="as"
          />
        </div>
      )
    }}
  </Story>
</Canvas>
