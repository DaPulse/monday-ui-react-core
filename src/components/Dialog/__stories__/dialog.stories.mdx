## Overview

Dialog is a positing engine, the purpose of this component is to position an element next to a different element (on top of near by elements);
We use this as our infra for the following components

- Tooltip
- Menu
- Menu Button

We use [PopperJS](https://popper.js.org/) behind the scenes. The `Dialog` component is a wrapper component it holds no UI logic.
