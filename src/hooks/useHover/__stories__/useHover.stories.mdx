## Overview

Attaches a listener for <code>mouseover</code> and <code>mouseout</code> DOM events on a specific element, returning <code>isHovered</code> state.

<Canvas>
  <Story name="Overview">
    {() => {
      const [hoverRef, isHovered] = useHover();
      return (
        <div ref={hoverRef} className="hooks-reference-element">
          Hover me! {isHovered ? "Boom!" : undefined}
        </div>
      );
    }}
  </Story>
</Canvas>
